# Sistema de Gestión de Libros Electrónicos

Este proyecto está desarrollado en Go con enfoque funcional. Permite registrar y consultar libros electrónicos desde consola.

## Estructura
- `cmd/`: Punto de entrada
- `internal/book/`: Libros
- `pkg/storage/`: Almacenamiento


## Instalación local
```bash
git clone https://github.com/TU_USUARIO/sistema-gestion-ebooks.git
cd sistema-gestion-ebooks
go run ./cmd
